# ⚡ n8n Automation Workflows

> Production-ready n8n workflow collection built from real-world 
> CRM & business automation projects.

## 🚀 What's Inside

| Workflow | Description | Tools Used |
|----------|-------------|------------|
| HubSpot → Salesforce Sync | Auto-sync leads across CRMs | HubSpot, Salesforce, n8n |
| Lead Notification System | Slack alerts for new CRM leads | HubSpot, Slack, n8n |
| Webhook Handler | Generic webhook processor | REST APIs, n8n |
| Google Sheets Sync | CRM data export automation | Google Sheets, n8n |

## 🛠️ How to Use
1. Clone this repo
2. Open your n8n instance
3. Go to **Workflows → Import from File**
4. Select any `.json` file from this repo
5. Configure your credentials & activate

## ⚙️ Requirements
- n8n (self-hosted or cloud)
- Relevant API credentials 
  (HubSpot, Salesforce, Slack etc.)
